# projeto-linktree
# 📱 Linktree Pessoal - Estilo Minimalista

Este é um projeto de **Linktree Pessoal (Cartão de Visitas Digital)** desenvolvido do zero para consolidar meus aprendizados sobre estruturação semântica, o comportamento do Box Model e técnicas avançadas de responsividade em CSS.

O layout foi planejado com foco na experiência do usuário em dispositivos móveis, utilizando uma paleta de cores elegante em tons terrosos para fugir do padrão convencional de designs saturados e neon.

---

## 🎨 Design & Funcionalidades
- **Abordagem Mobile-First:** Perfeitamente otimizado para telas de smartphones e tablets, eliminando qualquer tipo de rolagem horizontal.
- **Efeitos Dinâmicos:** Transições e animações suaves (`transition` e `transform: scale`) aplicadas aos botões no momento do toque ou hover.
- **Identidade Única:** Foto de perfil redonda com máscara CSS e ícones sociais leves (otimizados para carregamento rápido).
- **Rodapé Inteligente:** Alinhamento dinâmico usando propriedades do Flexbox para travar o footer sempre na base da tela.

---

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura limpa e semântica (`<header>`, `<main>`, `<footer>`, `<a>`).
- **CSS3:** Flexbox para alinhamentos, pseudo-classes (`:hover`), gerenciamento de sombras (`box-shadow`), tipografia e manipulação do Box Model.
- **Git & GitHub:** Versionamento de código e hospedagem pública.

---

## 🧠 Aprendizados neste Projeto
Neste repositório, pratiquei a resolução de problemas reais de layout:
1. Como evitar o vazamento de elementos para as laterais usando `box-sizing: border-box`.
2. Como posicionar elementos no final da página de forma elegante usando `margin-top: auto` combinado com o Flexbox.
3. A importância de escolher ícones com tamanhos de arquivo extremamente baixos (na faixa de 5KB) para garantir performance em redes móveis instáveis.
